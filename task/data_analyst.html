Data Analyst ‚Äî Internship Tasks

Objective, Features, Technologies and tasks to learn Data Analyst fundamentals and practical skills.

Objective

Train interns in data cleaning, exploratory analysis, visualization, and actionable business reporting using real-world datasets. Tasks are designed to build practical skills used in Alfido Tech projects.

Features

  • Hands-on analysis using Pandas/Excel/SQL
  • Cleaner, visualizations and executive summaries
  • Storytelling with data and actionable recommendations
  • Use of real datasets hosted on Kaggle

Tools

Python (Pandas / NumPy) Jupyter / Colab Excel / Google Sheets Tableau / Power BI (optional) SQL

Beginner Level Tasks

  • Download any of the provided Kaggle datasets and open it in Jupyter or Excel.
  • Report basic summary statistics (mean, median, mode, missing values) and data types.
  • Create 3 basic visualizations (bar chart, line chart, pie or histogram) summarizing important columns.
  • Write a one‚Äëpage summary with 3 actionable insights for Alfido Tech.

Note: Complete any 3 tasks out of the 4 main tasks below.

Tasks (4)

Dataset

https://www.kaggle.com/datasets/bhanupratapbiswas/customer-behavior-analysis

Goal

Analyze customer transactions & behavior to identify segments, purchase patterns, and churn risks.

Requirements
  • Perform data cleaning and feature engineering
  • Segment customers (RFM or clustering) and profile each segment
  • Visualize purchase patterns and retention trends
  • Provide 5 actionable recommendations for Alfido Tech to improve engagement
Deliverables
  1. Notebook (Jupyter/Colab) with code and visualizations
  2. PDF report summarizing segments, findings and recommendations

Dataset

https://www.kaggle.com/datasets/bhanupratapbiswas/superstore-sales

Goal

Analyze sales performance across regions, categories and time. Identify best/worst performing products and seasonality.

Requirements
  • Clean and prepare sales & order data
  • Create KPIs: revenue, profit margin, average order value, conversion metrics (if available)
  • Build visualizations showing trends, top products, and regional performance
  • Suggest 5 tactical improvements for Alfido Tech‚Äôs sales or offerings
Deliverables
  1. Notebook with analysis and charts
  2. Dashboard (Tableau/PowerBI or notebook) or PDF summary

Dataset

https://www.kaggle.com/datasets/bhanupratapbiswas/website-traffic-analysis

Goal

Analyze website traffic logs to understand user journeys, top landing pages, bounce rates and referral sources.

Requirements
  • Parse and clean traffic / log data
  • Compute metrics: sessions, users, bounce rate, average session duration
  • Visualize user flows and top entry/exit pages
  • Recommend 5 optimizations to improve conversions for Alfido Tech
Deliverables
  1. Notebook with analysis & visualizations
  2. Short report with key insights and action items

Goal

Choose or collect a dataset relevant to Alfido Tech and perform an end-to-end analysis addressing a business question (e.g., internship conversion funnel, service usage patterns).

Deliverables
  1. Notebook + README
  2. Presentation (PPT/PDF) with findings

How to Submit Your Tasks

  1. For each task:
    • Create a separate document (DOC/PDF) with code links, screenshots, visualizations and a short executive summary (1 page).
  2. Upload:
    • Upload notebooks & files to GitHub; large assets to Google Drive. Make sure links are viewable.
  3. Submit:
    • Go to the Task Submission page and paste your links, clearly mentioning task numbers.

Tip: Start with EDA, keep notebooks clean, and provide clear business recommendations tailored to Alfido Tech.
